Simpatico

javajosh 2023

Site status: tech demo.

Non-technical, non-learner users will find very little to interest them, currently.

Advanced technical users will be interested in the github code. They may also be interested in Simpatico's motivating ideas.

The browser kata serve as a good reference for browser capabilities. A good adjunct to MDN and other good internet resources.

Early apps

An extremely simple word processor. This is a tiny word processor in an html file that uses contenteditable and localstorage as if it was a little filesystem. Your "document" never leaves your browser.

A simple chat client. This is also a single, tiny html file that lets you send strings anyone else on the site.

A primoridal browser hosted database in the same style.

More pressing is my desire to visualize a novel data-structure and algorithm that I have high hopes for. The rtree is a very useful data-structure for modeling many important application problems. Comfort with svg and animation is useful for making visualizations. We build up to that with svg events. We also explore the browser's dynamic interpretation of svg with svg-compose
