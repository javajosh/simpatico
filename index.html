Site status: unreleased tech demo. Non-technical, non-learner users will find very little to interest them, currently. Advanced technical users will be interested in the github code. The browser kata serve as a good reference for browser capabilities. A good adjunct to MDN and other good internet resources.

There are proof-of-concept, early versions of end-user apps. An extremely simple word processor. This is a tiny word processor in an html file that uses contenteditable and localstorage as if it was a little filesystem. Your "document" never leaves your browser. A simple chat client. This is also a single, tiny html file that lets you send strings anyone else on the site. A primoridal browser hosted database in the same style.

More pressing is my desire to visualize a novel data-structure and algorithm that I have high hopes for. The stree is a very useful data-structure for modeling many important application problems. Comfort with svg and animation is useful for making visualizations. We build up to that with svg events. We also explore the browser's dynamic interpretation of svg with svg-compose. Now there are acceptance tests. There is also a crypto kata (that became a library).
