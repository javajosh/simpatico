Simpatico

javajosh 2023

Simpatico is for you. One measure of the success of a system is how much value someone can create with it.

How about a simple word processor? This is a tiny html file that lets you edit localstorage as if it was a little filesystem. It is a good demonstration of html authoring standard and the power of focusing your work.

A simple chat client. This is also a single, tiny html file that lets you send strings to everyone connected. This is a good example of using simple browser crypto to do a good job of hiding data. (Please don't abuse this one).

Eventually I'd like to try my hand at building a visual database in the same style. Start with sqlite in wasm using the new browser file api as a foundation.

More pressing is my desire to visualize a novel data-structure and algorithm that I have high hopes for. The rtree is a very useful data-structure for modeling many important application problems. Comfort with svg and animation is useful for making visualizations. We build up to that with svg events. We also explore the browser's dynamic interpretation of svg with svg-compose
